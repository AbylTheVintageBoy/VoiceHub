VoiceHub's Public Beta Release!

VoiceHub revives the website version of a TTS software known as VoiceForge. VoiceForge is a application that allows users to use high quality Text-To-Speech (TTS) audio to websites, animations, or games using their game service. Other TTS websites make it hard for users to use their TTS versions on their ideas because of these 3 reasons:

  • Text-to-Speech software is complex
  • The voices sound corporate and boring
  • The audio distribution rights are expensive

However, VoiceHub brings back this old website to make it easy for developers to use VoiceForge TTS without the hassle of using the mobile app. Enter some text, select a voice, and code it onto your idea using Swift Engine's API!

So what if it's VoiceForge... you might think it's a copy of every other TTS website, but it's not. VoiceForge has made over 40 fun and entertaining voices to use for your ideas! You can test them out using the Demo page, and when your actually ready to add these voices into your idea, all you have to do is code the API into your idea.

The process might be a little compilicated, but then again, it'll all be for the best when you have fun, entertaining, and interesting voices on your idea that will help it stand out, and get more users amazed by what you did! If you only feel like goofing around with these voices, then use the Demo page for fun! VoiceHub will always be here, and ready for you to use.
